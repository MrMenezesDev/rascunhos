Gerador Interativo de Ladrilhos Hexagonais

Ferramenta interativa (py5) para desenhar padrões de arestas em um grid hexagonal e exportar tiles únicos (edges) para JSON, servindo de insumo para experimentos de Wave Function Collapse / mosaicos.

Controles principais: Mouse ESQ alterna a borda mais próxima; SHIFT+ESQ liga todas; CTRL+ESQ desliga todas; DIR limpa célula; S exporta padrões; R reseta; C imprime padrão da célula; + / - ajusta inner; 1 2 3 paletas; 0 reseta; X aleatórias; B/U ciclam paletas barras/cubo; P/O aleatórias separadas; Q/E rotaciona; L reseta rotação; K salva PNG; G debug; V/I visibilidade; F fit; H ajuda.

Exemplos de tiles exportados e capturas:

O arquivo tiles.json contém objetos {"id": n, "edges": [e0..e5]} descrevendo quais arestas estão ativas (1) em cada direção (0 topo, sentido horário).
